# CubeTransformation
A Unity 3D game demonstrating cube transformation, circular motion, and camera perspective effects.

## Features
- Smooth circular movement
- Dynamic cube scaling effect
- Camera perspective illusion
- Unity 3D environment
- Beginner-friendly game mechanics

## Technologies Used
- Unity
- C#
- Visual Studio
 
## Project Goal
This project explores object transformation, motion, and perspective in a 3D game environment using Unity
